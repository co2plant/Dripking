# Dripking
술과 여행을 즐기는 사람들을 위한 술 관련한 여행지 추천 사이트

## 프로젝트 소개
*Dripking*은 술과 여행을 좋아하는 사람들을 위해 다양한 술 관련 여행지를 추천해주는 웹사이트입니다. 사용자는 자신이 선호하는 술 종류와 여행지를 선택하여 맞춤형 추천을 받을 수 있습니다.

### 기술스택
- **언어**: Java 21
- **프레임워크**: Spring Boot 3.5.5
- **빌드 도구**: Gradle 8.10.2
- **Gradle JVM**: corretto-21

### 개발기간

### 개발환경
- OS : Windows 11
- IDE: IntelliJ IDEA

### 문서 관리
- 제품 요구사항과 프론트엔드 연동 기준은 별도 문서로 정리하며, API 변경 시 프론트엔드 호출부와 함께 확인합니다.
- 백엔드 변경 전에는 관련 컨트롤러, DTO, 서비스, 엔티티의 요청/응답 계약을 먼저 확인합니다.
- Docker 기반 배포와 운영 환경 변수 기준은 `docs/deployment.md`를 따릅니다.
