# Dripking Product Overview

## 문서 목적

이 문서는 Dripking이 어떤 문제를 해결하는 프로젝트인지 설명하기 위한 제품 소개 문서입니다. 실행, 배포, 환경 변수 같은 운영 정보보다 서비스의 목적, 사용자 흐름, 백엔드가 담당한 구현 범위를 먼저 보여주는 데 초점을 둡니다.

## 제품 한 줄 소개

Dripking은 술과 여행을 좋아하는 사용자가 술, 양조장, 여행지를 탐색하고 관심 항목을 여행 일정으로 연결할 수 있게 돕는 여행 계획 웹 서비스입니다.

## 해결하려는 문제

술을 테마로 여행을 준비할 때 사용자는 보통 여러 정보를 따로 확인해야 합니다.

- 어떤 술이 어느 양조장 또는 증류소에서 만들어지는지
- 해당 생산지가 어느 여행지와 연결되는지
- 관심 있는 장소를 실제 여행 일정에 어떻게 배치할지
- 다른 사용자의 리뷰나 인기 신호를 어떻게 참고할지

Dripking은 이 과정을 "콘텐츠 탐색 -> 위시리스트 저장 -> 여행 일정 구성"이라는 하나의 흐름으로 연결합니다.

## 대상 사용자

| 사용자 | 필요 |
| --- | --- |
| 술과 여행을 함께 즐기는 일반 사용자 | 술, 양조장, 여행지를 쉽게 탐색하고 저장 |
| 위스키, 맥주, 전통주 등 특정 주류에 관심 있는 사용자 | 카테고리와 생산지를 기준으로 콘텐츠 탐색 |
| 양조장/증류소 방문을 계획하는 여행자 | 관심 항목을 실제 일정으로 구성 |
| 관리자와 콘텐츠 운영자 | 술, 양조장, 여행지, 리뷰, 사용자 상태 관리 |

## 핵심 사용자 흐름

1. 사용자는 술, 양조장, 여행지 목록과 상세 정보를 탐색합니다.
2. 검색, 필터, 정렬, 인기 콘텐츠를 통해 관심 있는 항목을 찾습니다.
3. 관심 항목을 위시리스트에 저장합니다.
4. 위시리스트 항목 또는 직접 입력한 장소를 여행 일정에 추가합니다.
5. 로그인 사용자는 서버에 위시리스트와 여행 계획을 저장합니다.
6. 사용자는 술, 양조장, 여행지에 리뷰를 작성하거나 부적절한 리뷰를 신고할 수 있습니다.
7. 관리자는 콘텐츠, 사용자, 신고 리뷰, 인기 데이터를 관리합니다.

## MVP 범위

MVP 1은 추천/탐색과 일정 관리 중에서 일정 관리 경험을 우선합니다. 탐색한 콘텐츠가 위시리스트를 거쳐 여행 일정으로 이어지는 흐름이 핵심입니다.

포함 범위:

- 회원가입, 로그인, JWT 인증
- 술, 양조장, 여행지 목록/상세/검색/필터/정렬
- 인기 여행지와 인기 술 추천
- 위시리스트 저장
- 여행과 세부 일정 관리
- 리뷰 작성, 수정, 삭제, 신고
- 관리자 콘텐츠 관리
- 관리자 리뷰 검수
- 관리자 대시보드와 인기 신호 확인
- S3-compatible storage 기반 콘텐츠 이미지 업로드

제외 또는 향후 범위:

- AI 여행 코스 자동 생성
- 크롤링 또는 자동 외부 데이터 수집
- 개인화 추천 알고리즘
- 소셜 로그인
- 알림
- 예약 시스템
- 다국어 UI

## 백엔드 구현 범위

백엔드는 서비스 데이터와 비즈니스 규칙을 담당합니다.

| 영역 | 구현 내용 |
| --- | --- |
| 인증 | Spring Security, JWT, 사용자/관리자 권한 분리 |
| 카탈로그 | 술, 양조장, 여행지, 국가, 도시, 카테고리 API |
| 위시리스트 | 로그인 사용자 위시리스트 저장과 조회 |
| 여행 계획 | `Trip`과 `Plan` 기반 여행 일정 CRUD |
| 리뷰 | 대상별 리뷰 작성, 수정, 삭제, 신고 |
| 관리자 | 사용자 관리, 콘텐츠 관리, 리뷰 검수, 대시보드 |
| 인기 점수 | 사용자 상호작용 이벤트를 기반으로 인기 콘텐츠 계산 |
| 이미지 | 관리자 콘텐츠 이미지 업로드와 렌더링 URL 관리 |

## 제품 차별점

- 술 상품, 양조장, 여행지를 하나의 여행 도메인으로 연결합니다.
- 단순 추천에서 끝나지 않고 관심 항목을 일정에 바로 반영합니다.
- 비로그인 사용자는 로컬 저장으로 먼저 계획을 만들 수 있고, 로그인 후 서버 저장 흐름으로 이어질 수 있습니다.
- 인기 콘텐츠는 단순 클릭 수가 아니라 조회, 위시리스트 추가, 일정 추가 같은 여러 상호작용을 점수화합니다.
- 관리자 화면과 API를 통해 콘텐츠 품질, 리뷰 신고, 인기 데이터를 함께 관리합니다.

## 기술 요약

- Java 21
- Spring Boot 3.3.4
- Spring Security
- JWT
- Spring Data JPA
- H2 for local development
- PostgreSQL for production target
- S3-compatible object storage
- Springdoc OpenAPI
- JUnit 5
